# 清单目录

本目录保存结构化设备、网络、强电和 RS-485 清单。

规划文件：

- `devices.csv`：设备、位置、协议、优先级、数量、状态和备注。
- `network_ports.csv`：弱电端口、房间、配线架、网络分区和施工备注。
- `power_socket_points.csv`：固定设备用电、插座点位、UPS 边界和智能插座禁区。
- `rs485_cable_runs.csv`：RS-485 候选线路、拓扑、施工估算长度和预留长度。
- `ip_plan.md`：VLAN、静态地址预留和主机名。
- `rs485_channels.md`：RS-485 网关端口、总线走线、串口参数、从机地址和终端电阻规则。
- `purchase_candidates.md`：硬件候选和决策状态。
- `docs/current/首期硬件型号与购买链接建议_V1.md`：具体型号、粗略价格区间和京东/淘宝搜索链接。

不要在这里保存密码、API key、令牌、VPN 密钥或私人账号恢复资料。

当前文件：

- `purchase_candidates.md`：首期硬件候选和采购优先级。
- `../docs/current/首期硬件型号与购买链接建议_V1.md`：当前型号级推荐和购买链接。
- `hardware_decision_matrix.md`：主控、网络生态、交换机、AP、Zigbee、RS-485 和 UPS 取舍矩阵。
- `contractor_wiring_checklist.csv`：给施工队使用的布线和安装条件清单。
- `network_ports.csv`：施工阶段弱电端口表。
- `power_socket_points.csv`：机柜、工作站、P2S、影音/投影、窗帘、水路设备、暖通、加湿和入户设备的强电/插座复核表。
- `ip_plan.md`：首期 VLAN/IP/静态地址规划。
- `rs485_channels.md`：RS-485 通道拆分和调试规则。
- `rs485_cable_runs.csv`：日立、格恩和未来预留通道的 RS-485 线长设计估算。
- `devices.csv`：首期设备总清单。

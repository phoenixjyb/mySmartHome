# 给设计师的点位落图输入（V1）

**日期：** 2026-07-04
**用途：** 作为业主侧给设计师的输入，指导水电开槽前的强电、弱电、设备、传感器、安全和检修点位落图。

## 1. 我们的总体建议

本项目不按第三方全屋智能厂家整包交付，而是由装修队和各设备厂家先交付“可调试条件”，业主侧后续负责 Home Assistant、Zigbee、网络、RS-485、自动化和软件调试。

设计图纸现在需要优先解决：

1. 固定设备有没有电源；
2. 需要联网的设备有没有网线或空管；
3. 后期不可补的线、管、底盒、检修口是否已落图；
4. 衣帽间机柜、P2S、柜体、桌面、窗帘盒和影音柜是否会遮挡点位；
5. 每根线、每个端点是否能编号、测试、拍照和验收。

无线传感器不需要逐个拉线，但水浸、安全报警、固定供电 CO2、存在传感器、电动阀、排风和窗帘电机相关的电源、空间和检修条件必须提前画清楚。

## 2. 请设计师输出的图纸

建议至少输出以下 6 类图。若图纸数量需要合并，也必须用图例区分强电、弱电、PoE、RS-485、传感器、预留空管和检修口。

| 图纸 | 必须包含 |
|---|---|
| 强电/插座图 | 固定设备插座、专用插座、窗帘电机 220V、智能开关零线和深底盒、总水阀候选电源、燃气/CO 报警电源候选 |
| 弱电/网络图 | Cat6A 网线、AP、PoE Zigbee、影音柜、投影、入户门、桌面网口、机柜回线、配线架编号 |
| RS-485/暖通弱电图 | 日立空调、格恩中央加湿各自独立 RS-485 管线；未来水表/电表/其他设备预留空管 |
| 天花/吊顶设备图 | AP1、AP2、PoE Zigbee、投影、幕布盒、窗帘盒、空调出回风口、检修口、烟感候选点 |
| 水路/安全点位图 | 次卫洗衣、厨下净水/RO、主卫/次卫、水浸、总水阀、电动阀候选、燃气/CO 报警 |
| 柜体/家具遮挡复核图 | 衣帽间机柜、P2S 台面、书房工作站、客厅影音柜、厨下柜、主卧阳台书桌、窗帘盒检修 |

## 3. 已确认的业主侧决策

| 项目 | 建议/决定 |
|---|---|
| 弱电中心 | 衣帽间作为网络和自动化设备间 |
| 机柜 | 12U 优先落地网络机柜，不做壁挂机柜 |
| 主控 | Apple Silicon Mac mini 作为首期 Home Assistant 主控，放衣帽间机柜附近 |
| UPS | 只给光猫、路由/防火墙、交换机、Mac mini、Zigbee/MQTT/RS-485 网关等低功耗控制设备供电 |
| 网线 | 全屋以太网默认 Cat6A，星形回衣帽间配线架 |
| AP | 首期只做 AP1、AP2 两个有线 PoE AP，不再预留 AP-R |
| Zigbee | PoE/Ethernet Zigbee 协调器放中央开放高位，不放金属机柜 |
| RS-485 | 日立空调和格恩中央加湿各走独立通道，不共线，不与强电共管 |
| 洗衣机 | 固定在次卫生间 |
| P2S | Bambu P2S 放衣帽间，独立台面和独立插座，不接机柜 UPS/PDU |
| 窗帘 | 每组窗帘都按电动和可接入智能系统设计，首选 Zigbee，RS-485/干接点为备选 |
| 照明 | 普通房间用智能开关/继电器，保留实体开关；客厅/餐厅局部考虑调光 |
| 智能开关 | 所有开关盒预留零线、地线和加深 86 底盒 |
| 传感器 | 水浸、安全报警和固定供电传感器条件先落图；普通 Zigbee 电池传感器后装 |

## 4. 强电/插座图输入

强电图不只画普通插座。请把以下固定设备和候选设备落图，并标注高度、回路/漏保/接地要求和是否可能被柜体遮挡。

| 区域 | 必须落图 |
|---|---|
| 衣帽间机柜 | 机柜/UPS 输入独立带地墙插；机柜 PDU；通风和检修空间 |
| 衣帽间 P2S | P2S/AMS 独立带地插座组；不接机柜 UPS/PDU；可选通风/排风电源 |
| 衣帽间清洁/备用 | 吸尘器充电插座；高功率/服务备用插座 |
| 书房 | 3090 工作站桌位多位带地插座组，按高功率桌面设备考虑 |
| 儿童房/次卧 | 书桌/电脑候选插座组 |
| 主卧阳台 | 书桌/阅读/多电脑候选插座组，避开窗帘和柜体 |
| 客厅影音 | 影音柜多位插座、投影仪电源、幕布电源、影音管同点位复核 |
| 次卫 | 洗衣机/烘干候选插座、防潮、接地、漏保、排水关系 |
| 厨房/厨下 | 水槽、RO/净水、洗碗机、厨下设备插座，必须可检修 |
| 主卫/次卫 | 智能马桶、镜柜、排风、浴室柜下电源 |
| 窗帘盒 | 每组窗帘盒 220V 常电；双层窗帘按两台电机预留 |
| 开关盒 | 每个智能开关位置预留零线、火线、灯控线、地线和加深 86 底盒 |
| 总水阀候选 | 电动总水阀电源、手动操作空间和检修空间 |
| 安全报警 | 燃气报警/一氧化碳报警若采用固定供电款，需落电源点 |
| CO2/存在候选 | 若采用固定供电款，客餐厅、书房、卫生间等候选点需落电源点 |

## 5. 弱电/网络图输入

所有以太网线星形回衣帽间机柜，不串接、不中途接头。每根线两端编号，完工后提供测试结果和隐蔽工程照片。

| 点位 | 建议 |
|---|---|
| AP1 | 左侧中央走廊吊顶/可暗线顶面，1 根 Cat6A，PoE |
| AP2 | 客厅边吊/空调出风口附近可施工区域，1 根 Cat6A，PoE，允许偏置但避开风管、出风口、灯具和幕布盒 |
| PoE Zigbee | 中央开放走廊/高位，1 根 Cat6A，PoE，设备明装或底座遮线 |
| 书房 | 2 根 Cat6A 到书桌/3090 工作站位置 |
| 儿童房/次卧 | 2 根 Cat6A 到书桌候选位置 |
| 主卧阳台 | 1 根 Cat6A 必选；第二根 Cat6A 或空管可选 |
| 客厅影音柜 | 默认 2 根 Cat6A；受限时至少 1 根 Cat6A + 1 根可穿线空管 |
| 投影位置 | 1 根 Cat6A 备用，并配影音穿线管 |
| 入户高位 | 1 根 Cat6A，PoE 摄像头候选，需避开邻居隐私 |
| 入户低位 | Cat6A 优先，至少空管 + 拉线，门铃候选 |
| 衣帽间 Mac mini/NAS/RS-485 网关 | 机柜内配线架到交换机跳接；设备位和供电同图复核 |
| 客厅阳台 | 1 根备用 Cat6A 或 Cat6，作为未来设备预留 |
| P2S 台面 | 可选 1 根 Cat6A；P2S 首期可用 Wi-Fi |

## 6. RS-485/暖通弱电图输入

RS-485 不等同于以太网，不能接普通交换机，不能和普通网口混标。

| 通道 | 设备 | 施工输入 |
|---|---|---|
| Port 1 | 日立中央空调转换器 | 独立弱电管 + 厂家确认的屏蔽 RS-485 双绞线，到衣帽间 RS-485 网关 |
| Port 2 | 格恩中央加湿 | 独立弱电管 + 厂家确认的屏蔽 RS-485 双绞线，到衣帽间 RS-485 网关 |
| Port 3 | 未来 Modbus 电表/水表/计量设备候选 | 预留空管和拉线条件；电动总水阀不默认归入 RS-485 |
| Port 4 | 未来其他设备候选 | 预留空管和拉线条件 |

要求：

- 日立和格恩不得共线、不得共管；
- RS-485 不得与强电同管；
- 图上标注 A/B/GND/屏蔽层、端点位置、检修口和线号；
- 厂家未确认线型前，至少先确认独立管路和拉线条件。
- 电动总水阀默认按强电/水路安全点位处理，预留电源、检修和手动操作空间；只有后续明确采购带 RS-485/Modbus 的阀门控制器时，才接入 RS-485。

## 9. 柜体和家具遮挡复核

以下位置必须在柜体深化时复核，不能只在平面图上画点：

- 衣帽间机柜门开启、PDU/UPS、散热、线缆进出和检修；
- P2S 台面、开门、AMS、耗材、维护、清洁和散热空间；
- 书房 3090 工作站插座、网口和散热，不能被桌腿/柜体挡住；
- 客厅影音柜插座、网口、穿线孔和散热；
- 厨下插座、水浸、净水设备和总水阀不能被橱柜永久封死；
- 主卧阳台桌面插座/网口不能被窗帘、柜体、门套遮挡；
- 每组窗帘盒必须能放电机并可检修。

## 10. 本轮不要求设计师做的事

- 不要求给每个 Zigbee 电池传感器拉线；
- 不再预留 AP-R/AP3；
- 不要求把所有卧室都做毫米波存在传感器；
- 不要求每个房间都上 CO2；
- 不要求 P2S 强制外排，但要保留可选通风/排风条件；
- 不要求施工队负责 Home Assistant、Zigbee2MQTT、MQTT、自动化软件调试。

## 11. 验收输入

封墙、封顶、封柜前，设计师和施工队需要交付以下证据：

1. 强电点位竣工图；
2. 弱电点位竣工图；
3. 每根网线两端编号和测试结果；
4. RS-485 A/B/GND/屏蔽层、路径和端点照片；
5. 隐蔽工程照片，包含管线走向、线号、底盒、检修口；
6. 衣帽间机柜供电、接地、通风和弱电进线照片；
7. 窗帘盒电源、电机侧和检修口照片；
8. 厨下、次卫洗衣、总水阀、水浸候选点的可检修照片；
9. 任何现场变更必须回写到图纸，不接受只口头说明。

## 12. 可以直接发给设计师的话

```text
这次请按新版平面图和拆旧后的现场条件，先帮我们完成强电/插座图、弱电/网络图、RS-485/暖通弱电图、天花设备图、水路/安全点位图、柜体遮挡复核图。

我们的原则是：所有以太网星形回衣帽间机柜；以太网默认 Cat6A；日立空调和格恩中央加湿各走独立 RS-485 管线；强电、弱电、RS-485、影音管不得混管；每根线和每个端点必须编号、测试、拍照。

请优先锁定水电开槽后难以补救的内容：衣帽间机柜和 P2S、AP1/AP2、PoE Zigbee、书房/儿童房/主卧阳台桌面网口和插座、客厅投影/幕布/影音柜、次卫洗衣、厨下水路、总水阀候选、每组窗帘电机、智能开关零线和深底盒、日立/格恩 RS-485、入户门预留。

无线 Zigbee 传感器可以只画安装区域，不需要逐个拉线；但水浸位置、安全报警位置、固定供电 CO2/存在传感器、电动阀、排风和窗帘电机相关电源需要进入图纸。

如果现场因为结构、柜体、吊顶、物业或设备安装冲突无法实现，请先反馈，不要现场直接取消或改成无线替代。
```

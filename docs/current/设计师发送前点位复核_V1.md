# 设计师发送前点位复核（V1）

**项目：** mySmartHome / smartHouse  
**日期：** 2026-06-28  
**用途：** 在发给设计师和施工队之前，确认所有安装点位是否可画入施工图、是否还有歧义。

## 1. 审计结论

当前方案可以发给设计师作为基础，但不建议直接让施工队照文字开工。需要设计师在图纸上进一步明确以下内容：

1. 储物间机柜的实际墙面、尺寸、门开启方向、通风路径和插座位置；
2. AP1、AP2、AP-R、ZIGBEE-POE 的吊顶精确点位；
3. Bedroom1 / Bedroom2 书桌位置、网口高度、插座组位置；
4. 投影、幕布、影音柜之间的相对位置和影音管路径；
5. 储物间/新图衣帽间 P2S 独立台面、插座、维护空间和可选通风/外排预留；
6. 入户门高位摄像头和低位门铃预留方式；
7. 日立和格恩 RS-485 独立管线路径；
8. 每组窗帘电机侧、是否双层、弱电空管终点；
9. 每个智能开关底盒是否有零线、底盒深度和多控关系；
10. 水浸点、总水阀、净水/软水/热水器/加湿设备位置；
11. 所有固定设备的插座点位、功率确认、UPS 边界和强弱电同图复核。

## 2. 本次消除的歧义

### 2.1 Zigbee 点位

Zigbee 本身是无线协议，但本项目的协调器安装方式应理解为：

```text
Zigbee 设备  <--无线 Zigbee-->  PoE/Ethernet Zigbee 协调器  <--有线网线/PoE-->  交换机/Home Assistant
```

因此施工队需要做的是：

- 在中央走廊开放位置预留 1 根 Cat6 网线；
- 该网线回储物间配线架；
- 未来接 PoE/Ethernet Zigbee coordinator；
- 不把协调器放储物间金属机柜内；
- 不要求每个 Zigbee 传感器单独布线。

### 2.2 AP-R

原文写 `0-1` 容易让施工队理解为“可完全不做”。现在建议改为：

- **至少预留一根空管到 Bedroom3 外走廊或吊顶检修区；**
- 可以不穿网线、不采购 AP3；
- 但如果吊顶封闭后无法补线，则应直接穿 1 根 Cat6/Cat6A。

### 2.3 AV-DATA

原文写 `1-2`。给施工队建议改为：

- 客厅影音柜默认预留 **2 根 Cat6**；
- 如果图纸/成本/管径不允许，至少 1 根 Cat6 + 1 根可穿线空管；
- 两根线都回储物间配线架，未使用的不跳接即可。

### 2.4 ENTRY-L

原文写 `Cat6 或空管`。给施工队建议改为：

- 入户门旁低位至少预留 **空管 + 拉线**；
- 如不影响门套/墙体/物业要求，优先直接穿 1 根 Cat6；
- 最终是否安装门铃以后决定。

### 2.5 RS-485 线型

目前日立和格恩的最终线型仍需厂家确认。为了不阻塞装修：

- 先确认 **独立弱电管路径**；
- 管内预留拉线；
- 若厂家已给明确线型，则按厂家指定屏蔽双绞线施工；
- 未确认前不得把日立和格恩放同一管同一线；
- 强电、空调电源线和 RS-485 不共管。

### 2.6 交换机 16/24 口

文档中仍保留 16 口可用，但本轮复核建议：

- 图纸和机柜空间按 **24 口配线架 + 24 口交换机余量** 预留；
- 如果最后买 16 口交换机，也不影响施工；
- 机柜深度、PDU、理线和散热按更大设备预留。

## 4. 发给设计师时的请求

建议直接给设计师提出以下要求：

1. 请把本清单中所有点位落到平面图/天花图/弱电图/强电图上；
2. 每个弱电点标编号，编号与 `inventory/contractor_wiring_checklist.csv` 一致；
3. 对 AP、Zigbee、投影、窗帘、入户门和衣帽间 P2S/机柜单独出局部标注；
4. 储物间单独出机柜立面/设备布置示意；
5. 封墙前必须由业主、设计师、施工负责人三方复核；
6. 如果某个点位因为结构、吊顶、物业或家具冲突无法实现，先反馈，不要现场自行改成无线或取消。

## 5. 仍需业主确认

- Bedroom1 和 Bedroom2 最终书桌位置；
- Bedroom1 是否预留到储物间的大口径 HDMI/USB/光纤空管；
- 投影、幕布、影音柜最终位置；
- 衣帽间 P2S 台面、AMS/耗材位置、独立插座、维护空间和可选通风/外排路径；
- 入户门高位摄像头、低位门铃、或只预留；
- 是否直接按 24 口交换机空间预留；
- 窗帘是否双层、每组电机在哪一侧；
- 是否安装电动总水阀；
- 日立转换器和格恩控制器实际安装位置。
- RTX 3090、P2S、AV/投影、洗衣/水处理、窗帘、机柜等设备的插座数量、功率和是否需要专用供电。

## 6. 不建议发给施工队模糊执行的项目

以下项目必须先由设计师或厂家明确：

- “AP 大概放走廊”：必须有吊顶点位；
- “Zigbee 放中间”：必须有一根 PoE 网线终点；
- “RS-485 到储物间”：必须分日立和格恩两条独立路径；
- “门口预留”：必须说明高位摄像头和低位门铃分别预留；
- “窗帘智能化”：必须说明每组窗帘的电机侧、强电和弱电空管；
- “智能开关”：必须说明零线、底盒深度和多控关系；
- “P2S 放衣帽间”：必须说明台面、开门/维护空间、独立插座、不接机柜 UPS/PDU、烟雾/温湿度监测，以及可选通风/外排预留。
- “设备用电”：必须说明插座点位、功率确认、是否进 UPS、是否可被智能插座控制。

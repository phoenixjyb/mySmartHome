# 明日设计师沟通要点（2026-07-05）

**用途：** 明天和设计师对进度时使用。目标是在拆旧基本完成后，尽快锁定水电开槽前必须落图的强电、弱电、RS-485、天花设备、水路安全和柜体遮挡条件。

## 1. 明天要先讲清楚的总原则

1. 本项目没有第三方智能厂家整包交付，施工侧需要交付“可调试条件”，后续由业主侧负责 Home Assistant、Zigbee、网络、RS-485、自动化和软件调试。
2. 设计师需要把点位落到施工图，不只是在微信群里口头确认。
3. 优先锁定后期难补的内容：电源、网线、空管、底盒、检修口、柜体遮挡和编号。
4. 所有以太网默认 Cat6A，星形回衣帽间机柜。
5. RS-485 本轮刚性只针对日立中央空调和格恩中央加湿；未来电表/水表/计量设备只做可选预留，不作为本轮必须施工项。
6. 电动总水阀默认按强电/水路安全设备处理，预留电源、检修和手动操作空间，不默认接 RS-485。

## 3. 明天要给设计师的明确输入

### 3.1 衣帽间

- 衣帽间是弱电中心和自动化设备区。
- 落地 12U 网络机柜优先，不做壁挂。
- 机柜需要独立墙插给 UPS 输入，UPS 后接机柜 PDU。
- Mac mini、路由/防火墙、交换机、RS-485 网关在机柜或机柜旁。
- Bambu P2S 放衣帽间，但和机柜分区，独立台面、独立插座，不接机柜 UPS/PDU。
- P2S 只做可选通风/排风预留，PLA/PETG 不要求强制外排。

### 3.2 网络和弱电

- 全屋以太网默认 Cat6A。
- 所有网线星形回衣帽间配线架，不串接、不中途接头。
- AP 只做 AP1/AP2 两个 PoE 点位，不再预留 AP-R。
- PoE Zigbee 协调器放中央开放高位，不进金属机柜。
- 书房保留 2 根 Cat6A；儿童房/次卧保留 2 根 Cat6A；主卧阳台如作书桌，至少 1 根 Cat6A。
- 客厅影音柜默认 2 根 Cat6A；投影位置 1 根备用 Cat6A + 影音穿线管。

### 3.3 RS-485

- 日立中央空调和格恩中央加湿必须分通道、分管、分线。
- 日立走 RS-485 网关 Port 1，格恩走 Port 2。
- 线型按厂家确认的屏蔽 RS-485 双绞线；未确认前先做独立管路和拉线条件。
- Port 3 只作为未来 Modbus 电表/水表/计量设备预留，不要求现在接入任何表。
- 不假设小区正式电表/水表可被业主本地读取。
- 电动总水阀不默认接 RS-485，按水路安全设备处理。

### 3.4 强电和开关

- 每个智能开关盒预留零线、地线和加深 86 底盒。
- 窗帘盒每组预留 220V 常电，双层窗帘按两台电机考虑。
- 客厅/餐厅可以考虑局部调光，其它房间先按普通智能开关处理。
- 3090、P2S、洗衣、厨房、卫浴、空调、加湿、热水和总水阀等关键设备不得通过普通智能插座直接断电。

### 3.5 水路和安全

- 次卫洗衣机位置、进水、排水、地漏、防潮插座和水浸低位必须落图。
- 厨下净水/RO/水槽设备必须有可检修插座和水浸点。
- 总水阀位置必须确认；如果以后装电动阀，需电源、检修、手动操作空间。
- 燃气报警和一氧化碳报警按燃气公司/物业/产品说明落位。
- 烟感至少考虑客厅/走廊、衣帽间设备区、P2S 位，厨房外候选。

## 4. 明天要问设计师/施工队的问题

1. 光纤入户和原弱电箱位置在哪里？能否顺利回衣帽间机柜？
2. 衣帽间机柜和 P2S 是否能同时满足门开启、散热、检修和安全距离？
3. 客厅边吊/空调出风口附近是否能暗走 AP2 网线，设备是否能明装且不外露线？
4. PoE Zigbee 高位是否有合适位置，能否避开风管、金属、灯带驱动和大灯具？
5. 投影、幕布盒、影音柜和影音穿线管的相对位置是否已经确定？
6. 每组窗帘是单层还是双层，电机侧在哪边，检修怎么做？
7. 书房、儿童房/次卧、主卧阳台的桌位方向是否确定？
8. 次卫洗衣机、烘干候选、地漏和插座位置是否冲突？
9. 厨下净水/RO/总水阀是否会被橱柜永久封死？
10. 日立转换器和格恩控制器预计安装在什么位置，是否可检修？
11. 配电箱是否有空间给未来业主自装导轨电量表或监测模块？如果没有，本轮不强求。
12. 封墙、封顶、封柜前，谁负责拍照、编号、测试并回写图纸？

# 设计师点位参考方案（V1）

**日期：** 2026-07-04
**用途：** 给设计师作为点位落图参考。设计师应基于新版平面图、现场拆旧情况、柜体深化图和水电规范，把本方案转成正式强电、弱电、天花、暖通弱电、水路安全和柜体遮挡复核图。

## 1. 本方案的边界

本方案只给设计师和施工队需要提前预留的内容：

- 强电插座、常电、电源位置、回路和漏保条件；
- Cat6A 网线、PoE 点位、影音穿线管、弱电空管；
- RS-485/暖通弱电独立管线；
- 窗帘电机电源、电机侧、检修条件；
- 排风、电动阀候选、安全报警等需要固定电源或空间的点位；
- 机柜、P2S、厨下、次卫洗衣、总水阀等检修和柜体遮挡条件。

本方案不要求施工队处理以下内容：

- Zigbee 电池温湿度、水浸、门磁/窗磁、普通人体 PIR、场景按钮的逐点布线；
- CO2 固定点位；
- 独立场景面板专门开槽；
- Home Assistant、Zigbee2MQTT、MQTT、自动化和软件调试；
- 室内摄像头默认安装；
- 门窗执行器、电动开窗器或磁吸控制。

## 2. 设计总原则

1. 衣帽间是全屋网络和自动化中心，所有弱电星形回衣帽间机柜。
2. 以太网默认 Cat6A，强电、弱电、RS-485 和影音管不得混管。
3. Zigbee 终端多数无线；只有 PoE Zigbee 协调器需要 1 根网线回机柜。
4. 当前窗帘按单层窗帘方案，每组 1 台电机、1 个控制通道。
5. 窗帘电源是电机侧 220V 常电，不通过普通墙壁开关切断。
6. 场景面板不作为本轮施工必需项；后续如需要，可用无线按钮、厂家遥控、手机/语音/Home Assistant 面板替代。
7. 电池类传感器只需要保证能放置、能贴装、能检修，不需要施工队逐个拉线。
8. 成本优先：只保留高概率使用、封墙后难补的线管；纯候选、低概率、可后装的内容不默认预留。
9. 封墙、封顶、封柜前，所有线、管、盒、检修口和端点必须拍照、编号、测试。

## 3. 强电/插座参考方案

| 区域 | 设计师建议落图点位 | 施工说明 |
|---|---|---|
| 衣帽间机柜 | 机柜/UPS 输入独立带地墙插 1 组 | 给 UPS 输入，UPS 后接机柜 PDU；供光猫、路由/防火墙、交换机、Mac mini、RS-485 网关等低功耗设备 |
| 衣帽间 P2S | P2S/AMS 独立带地插座 1 组 | 不接机柜 UPS/PDU；与衣物、纸箱、溶剂、网络机柜分区 |
| 衣帽间清洁 | 吸尘器/清洁设备充电插座 1 组 | 与机柜和 P2S 分开 |
| 衣帽间备用 | 高功率/服务备用插座 1 组 | 不接 UPS；若未来接高功率设备需电工复核回路和通风 |
| 书房 | 3090 工作站桌位多位带地插座组 | 避免桌腿/柜体遮挡；不接 UPS；不得用普通智能插座切断工作站电源 |
| 儿童房/次卧 | 书桌/电脑候选插座组 | 按未来电脑、台灯、充电器预留 |
| 主卧阳台 | 书桌/阅读/多电脑候选插座组 | 避开窗帘、柜体、门套和日晒潮气 |
| 客厅影音柜 | 多位带地插座组 | 给功放、播放器、游戏机、小交换机等；保留散热和检修 |
| 客厅投影 | 投影位 220V 插座 | 与投影吊架、幕布位置、影音管同图复核 |
| 客厅幕布 | 电动幕布盒 220V 电源 | 只作为电动幕布供电和检修条件；不要求单独接入智能系统，不预留网线/RS-485 |
| 全屋窗帘盒 | 每组窗帘盒电机侧 220V 常电 | 单层窗帘，每组 1 台电机；接地、检修；电源不经过普通墙壁开关切断 |
| 次卫生间 | 洗衣机插座，烘干候选按需 | 防潮、接地、漏保、排水、地漏和柜体遮挡一起复核 |
| 厨房/厨下 | 净水/RO/厨下设备插座 | 必须可检修；不得被橱柜永久封死 |
| 主卫/次卫 | 智能马桶、镜柜、排风、浴室柜下电源 | 按卫生间分区规范、防潮、接地、漏保 |
| 总水阀候选 | 电动总水阀候选电源 | 预留电源、检修和手动操作空间；不默认接 RS-485 |
| 燃气/CO 报警 | 固定供电候选 | 位置和高度按燃气公司、物业、产品说明和规范确定 |
| 排风 | 主卫、次卫排风电源和实体开关 | 保留实体开关；后续可由湿度/存在联动 |

## 4. 弱电/网络参考方案

所有网线星形回衣帽间配线架，不串接、不中途接头。建议每根线两端编号。

| 编号 | 区域 | 参考点位 | 线缆/说明 |
|---|---|---|---|
| AP1 | 左侧中央走廊 | 吊顶或可暗线顶面 | 1 根 Cat6A，PoE；避开风管、金属、灯具和检修口 |
| AP2 | 客厅边吊/空调出风口附近可施工区域 | 允许偏置安装 | 1 根 Cat6A，PoE；避开风口、幕布盒、灯带驱动和大灯具 |
| ZIGBEE-POE | 中央走廊开放高位 | PoE Zigbee 协调器 | 1 根 Cat6A 或 Cat6，PoE；不放金属机柜内 |
| B1-DATA | 书房书桌/3090 位 | 桌面弱电点 | 2 根 Cat6A |
| B2-DATA | 儿童房/次卧书桌候选 | 桌面弱电点 | 2 根 Cat6A |
| MB-BAL-DATA | 主卧阳台书桌 | 桌面弱电点 | 至少 1 根 Cat6A；第二根或空管可选 |
| AV-DATA | 客厅影音柜 | 影音柜内 | 首期 1 根 Cat6A；第二根仅在同点位同管成本很低时可选 |
| PJ-DATA | 客厅投影 | 投影位置 | 投影可用 Wi-Fi 和影音管；Cat6A 不作为必需项，有条件再做备用 |
| BAL-DATA | 客厅阳台 | 备用 | 1 根备用 Cat6A 或 Cat6，未来设备候选 |

## 5. RS-485/暖通弱电参考方案

RS-485 不等同于以太网，不接普通交换机，不和普通网口混标。

| 通道 | 设备 | 设计师落图要求 |
|---|---|---|
| Port 1 | 日立中央空调转换器 | 独立弱电管 + 厂家确认的屏蔽 RS-485 双绞线，到衣帽间 RS-485 网关 |
| Port 2 | 格恩中央加湿 | 独立弱电管 + 厂家确认的屏蔽 RS-485 双绞线，到衣帽间 RS-485 网关 |
| Port 3 | 未来 Modbus 电表/水表/计量设备候选 | 只做空管/拉线候选，不作为本轮必须接表 |
| Port 4 | 未来其他设备候选 | 只做预留 |

要求：

- 日立和格恩不得共线、不得共管；
- RS-485 不得与强电同管；
- 图上标注端点、检修口、线号、A/B/GND/屏蔽层；
- 厂家未确认线型前，至少先锁定独立管路和拉线条件。

## 6. 天花/吊顶设备参考方案

| 点位 | 参考位置 | 说明 |
|---|---|---|
| AP1 | 左侧中央走廊顶面 | 优先服务书房、次卧、厨房/过道一侧 |
| AP2 | 客厅边吊或空调出风口附近可施工区 | 客厅无满吊顶时允许偏置；暗线出线、设备明装 |
| PoE Zigbee | 中央开放走廊高位 | 尽量居中，避开金属、强电驱动、电机和大灯具 |
| 投影 | 按新版平面图幕布位置反推 | 电源、吊架和影音管统一确认；投影网线降级为可选 |
| 幕布盒 | 客厅幕布位置 | 电动幕布电源、检修和吊顶关系确认；不单独接入智能系统 |
| 窗帘盒 | 每组窗帘 | 单层窗帘，电机侧、常电、检修方式必须标注 |
| 烟感候选 | 客厅/走廊、衣帽间设备区、P2S 位 | 独立报警优先；不依赖 Home Assistant |
| 排风 | 主卫、次卫，衣帽间/P2S 可选 | 主卫/次卫为重点；P2S 排风仅作为可选预留 |

## 7. 水路/安全参考方案

| 区域 | 参考点位 | 说明 |
|---|---|---|
| 次卫洗衣 | 洗衣机进水、排水、地漏、插座、水浸低位 | 水浸是电池传感器，不拉线；但必须留低位可放置空间 |

## 8. 房间级参考方案

### 8.1 衣帽间

- 落地 12U 网络机柜位置；
- 机柜/UPS 输入独立墙插；
- 弱电总进线、配线架、交换机、路由/防火墙、Mac mini、RS-485 网关空间；
- P2S 独立台面、独立插座、开门、AMS、耗材、废料、维护空间；
- P2S 不做墙面网口；如未来确实需要有线网络，可在衣帽间内直接接交换机或机柜内短跳线；
- 吸尘器充电插座；
- 高功率/服务备用插座；
- 机柜和 P2S 分区，避免衣物、纸箱、溶剂、胶水贴近设备；
- 烟感和温湿度位置只需保留安装空间，不要求逐个布线。

### 8.2 客餐厅

- AP2 点位；
- 客厅影音柜电源和 1 根 Cat6A；第二根网线只在同点位成本很低时可选；
- 投影电源、影音穿线管；投影网线不作为必需项；
- 幕布电源和检修；幕布盒不单独接入智能系统；
- 客厅阳台/大窗单层电动窗帘电机侧 220V 常电；
- 常规插座由设计师按家具和规范补齐；
- CO2 不作为固定点位，后续用普通插座桌面设备试点。

### 8.3 玄关/入户/过道

- PoE Zigbee 协调器中央高位 1 根网线；
- 入户高位摄像头和低位门铃本轮暂不预留；
- AP1 如设计师认为更适合过道顶面，可与 Zigbee 保持距离；
- 入户门磁为后贴电池设备，不需要布线。

### 8.4 书房

- 书桌/3090 工作站 2 根 Cat6A；
- 高功率桌面插座组；
- 单层窗帘电机侧 220V 常电；
- 温湿度、门窗磁、存在/人体等后装传感器不要求施工队布线；
- 不为场景面板专门开槽。

### 8.5 儿童房/次卧

- 书桌候选 2 根 Cat6A；
- 书桌插座组；
- 单层窗帘电机侧 220V 常电；
- 温湿度、门窗磁、人体传感器后装，不拉线。

### 8.6 主卧和主卧阳台

- 主卧阳台如作为书桌/阅读区，至少 1 根 Cat6A；
- 主卧阳台桌面多位插座组；
- 主卧/主卧阳台单层窗帘电机侧 220V 常电；
- 床头、阅读、充电等常规强电由设计师按家具补齐；
- 门窗磁、温湿度、人体传感器后装。

### 8.7 厨房

- 厨下净水/RO/水槽设备插座；
- 插座和水路必须可检修；
- 水浸传感器后装，但低位空间要保留；
- 燃气报警按规范和燃气公司要求落位。

### 8.8 主卫/次卫

- 次卫洗衣机固定位置、电源、进水、排水、地漏；
- 主卫/次卫排风实体开关；
- 智能马桶、镜柜、浴室柜下电源按规范落图；
- 水浸、电池人体、温湿度传感器后装，不逐点布线。

## 9. 给设计师可直接使用的文字

```text
请基于新版平面图和现场拆旧情况，按强电/插座、弱电/网络、天花设备、RS-485/暖通弱电、水路/安全、柜体遮挡六个图层落图。

本轮只要求施工队处理线、电、管、盒、孔、检修空间和编号测试。Zigbee 电池类温湿度、水浸、门磁/窗磁、普通人体 PIR、场景按钮不需要逐个布线。CO2 不作为固定点位。场景面板不作为本轮必须点位。

硬性要求：所有网线星形回衣帽间机柜，默认 Cat6A；AP1/AP2 和 PoE Zigbee 需要网线/PoE；日立空调和格恩加湿各自独立 RS-485 管线；强电、弱电、RS-485、影音管不得混管；封墙前必须拍照、编号、测试。

窗帘已确认单层。每组窗帘按 1 台电机和 1 个控制通道预留，电机侧提供 220V 常电、接地和检修空间，电源不通过普通墙壁开关切断。墙面控制不作为施工必需项，后续可用厂家遥控、无线按钮或智能系统控制。

如现场结构、柜体、吊顶、物业或设备安装条件导致某点无法实现，请先反馈，不要现场直接取消或改成无线替代。
```

## 10. 设计师需要回填确认的问题

1. 衣帽间机柜和 P2S 是否能同时满足门开启、散热、检修和安全距离；
2. 光纤入户能否顺利回衣帽间机柜；
3. AP1、AP2、PoE Zigbee 的精确顶面/高位位置；
4. 投影、幕布、影音柜和影音穿线管的相对位置；
5. 每组单层窗帘的电机侧、窗帘盒尺寸、检修方式；
6. 次卫洗衣、厨下净水、总水阀和水路设备是否被柜体遮挡；
7. 日立转换器和格恩控制器的安装位置和检修方式；
8. 书房、儿童房/次卧、主卧阳台桌位是否最终确定；
9. 所有强弱电、RS-485 和影音管的编号和隐蔽工程照片交付方式。

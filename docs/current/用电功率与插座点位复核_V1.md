# 用电功率与插座点位复核（V1）

**项目：** mySmartHome / smartHouse  
**日期：** 2026-06-28  
**用途：** 在设计定稿前，把设备功率、插座点位、回路边界、UPS 边界和智能控制禁区纳入图纸复核。

## 1. 结论

弱电点位只解决设备能否联网或接入系统；强电和插座点位决定设备能否安全、稳定、可维护地运行。设计师和电工需要在施工图中同时标出：

1. 每个固定设备的大致位置和插座高度；
2. 每个高功率设备的铭牌功率或厂家建议供电要求；
3. 哪些设备需要专用插座或独立回路；
4. 哪些插座需要防溅、防潮、接地、漏电保护和检修空间；
5. 哪些设备可以进 UPS，哪些严禁接入 UPS；
6. 哪些设备可以用智能插座监测，哪些设备不得用智能插座切断电源；
7. 预留插座是否会被家具、柜体、窗帘盒、设备平台或门套遮挡。

电工需要按现行规范、物业要求和现场配电容量完成最终回路、线径、断路器和漏保设计。本文只作为设计复核清单，不替代专业电气设计。

## 2. 原则

- 高功率设备先按铭牌功率和厂家安装要求确认，不按“平时用不了多少”估算。
- 潮湿区域、厨房、阳台、洗衣和水处理区域的插座必须考虑防溅、防潮、接地、漏电保护和检修。
- UPS 只给网络、主控、网关、低功耗传感/通信设备供电；不带 RTX 3090、P2S、空调、加湿、排风、热水、洗衣、厨房电器。
- 中央空调、中央加湿、燃气/热水、总水阀等关键设备不得用智能插座直接断电。
- 智能插座可以优先用于低风险负载的能耗监测，例如部分台灯、影音设备、非关键小电器；不要把它当作安全保护装置。
- 每组固定设备旁边至少留一个普通备用插座，避免未来全部依赖插排。
- 弱电点位和强电插座需要同图复核：有网口但无电源，或有电源但被家具挡住，都会导致点位不可用。

## 4. 需要设计师在图上增加的强电/插座标注

设计师需要把以下内容和弱电点位一起落图：

- 储物间/衣帽间机柜插座/PDU/UPS位置；
- Bedroom1、Bedroom2 书桌插座组；
- 客厅 AV 柜插座组、按新版平面图幕布位置协调的投影插座、幕布/投影相关电源；
- P2S 候选区插座组、排风设备插座、备用插座；
- 次卫生间洗衣/烘干/水路设备插座；
- 厨下净水/RO/软水/水浸相关预留；
- 卫生间智能马桶、镜柜、排风、传感器候选电源；
- 每组窗帘盒 220V 常电；
- 每个智能开关盒的零线和底盒深度；
- 电动总水阀候选位置的电源和检修空间；
- 入户高位/低位设备若不用 PoE，需要单独标出电源策略。

## 5. 不能交给智能系统直接断电的设备

以下设备不应通过普通智能插座直接切断电源：

- 中央空调主机、内机供电和原厂控制器；
- 中央加湿主机或原厂控制器；
- 燃气热水器、热水设备、带安全保护逻辑的设备；
- 总水阀、安防、烟雾、燃气、一氧化碳等安全相关设备；
- P2S 打印中途供电控制；
- 高功率厨房、卫浴、洗衣、烘干等设备；
- RTX 3090 工作站主供电。

这些设备可以考虑“状态监测、告警、联动提醒”，但不应把 Home Assistant 或智能插座作为唯一安全控制。

## 6. 给电工/施工队的交付要求

电工和施工队交付时需要提供：

1. 强电点位竣工图；
2. 每个固定插座的实际位置照片；
3. 机柜插座、PDU、UPS、接地和通风确认；
4. 高功率设备插座和回路说明；
5. 潮湿区域插座防护和漏保说明；
6. 智能开关盒零线、底盒深度和多控关系确认；
7. 窗帘盒电源和检修口照片；
8. P2S/排风插座和排风路径照片；
9. 厨下、水路、总水阀候选点的插座和检修空间照片；
10. 所有现场变更记录。

## 7. 仍需确认

- Bedroom1 RTX 3090 工作站的最终桌位、主机位置、显示器数量和插座数量；
- Bedroom2 是否确定预留台式机级别桌面电源；
- AV 柜设备数量和是否需要柜内通风；
- 投影和幕布最终型号、安装方式和电源点；
- P2S、AMS、排风风机和传感器的插座分组；若放衣帽间，必须独立于网络机柜供电和排风；
- 次卫洗衣机、烘干机、净水、软水、热水、加湿设备的最终型号；
- 是否安装电动总水阀；
- 门铃/摄像头是否采用 PoE，或需要额外电源；
- UPS 只覆盖哪些设备，电池续航目标是多少。

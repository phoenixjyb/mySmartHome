# 新平面图初读与智能点位影响（V1）

**项目：** mySmartHome / smartHouse  
**日期：** 2026-06-28  
**依据：** `references/newFloorPlan.png`  
**用途：** 根据新版平面布置图，先识别空间功能、家具设备位置和对智能家居点位的影响。本文不是最终施工图，不能直接替代弱电图、强电图、天花图或厂家深化图。

## 1. 初读结论

新版平面图比旧文字基线更接近实际装修方案，已经显示了书房、次卧、主卧、衣帽间、玄关、厨房、两个卫生间、客餐厅和三个阳台/外窗空间的家具布置。

对智能家居影响最大的变化是：

1. 原方案里的“储物间网络设备间”在新图中不再清晰独立出现，可能已经变成衣帽间/玄关柜/冰箱附近的柜体区域。这会直接影响机柜、弱电箱、UPS、Mac mini、交换机、RS-485 网关和散热维护。
2. 新图左侧明确出现“书房”，它比主卧更像 RTX 3090 工作站和固定办公位候选。原 `Bedroom1`、`Bedroom2`、`Bedroom3` 编号需要重新映射到新图房间名称。
3. 洗衣机位置出现在次卫生间附近，而不是阳台。水浸传感器、插座、排水、漏保和用电功率清单需要跟随调整。
4. 客厅有长柜、沙发、餐桌和疑似幕布/影音墙位置，投影、幕布、AV 柜、网口和影音管需要按新图重新确认。
5. 主卧阳台、次卧阳台都出现书桌/储物功能，客厅阳台仍是 P2S 外排的更可能候选，但图中没有明确画出 P2S 工作台。
6. 图中柜体很多，AP、Zigbee 协调器和传感器点位必须避开金属柜、厚柜体、风口、灯槽和检修口冲突。

因此，当前不建议马上冻结所有点位；应先让设计师基于这张新图确认网络设备区、办公主位、P2S、投影、洗衣/水路和厂家控制器位置。

## 3. 与当前基线的主要冲突

### 3.1 储物间/网络设备间冲突

当前文档假设存在独立 `Storage room`，用于放置 ONT、路由/防火墙、交换机、配线架、Mac mini、UPS、Zigbee 网络路径和 RS-485 网关。

新图里没有明显独立储物间。最接近的位置是玄关旁衣帽间/柜体区，但该区域同时出现衣柜和冰箱。这个位置如果要承载网络设备，必须满足：

- 机柜或弱电柜有明确宽、高、深和开门空间；
- 至少有独立接地插座、PDU、UPS 放置位；
- 有散热路径，不能封闭在衣物柜内；
- 弱电线、RS-485 管、入户光纤/网线可集中到达；
- 后期维护不需要拆衣柜、不影响冰箱散热；
- 不把 PoE Zigbee 协调器放进金属柜或密闭柜。

若这些条件无法满足，需要让设计师另找网络设备区，或者把弱电集中柜和主控主机分离设计。

### 3.2 Bedroom 编号需要重新映射

旧清单使用 `Bedroom1/Bedroom2/Bedroom3`。新图中文房间名更清楚，但对应关系未确认。

建议先暂定：

| 旧编号 | 新图候选 | 判断 |
|---|---|---|
| Bedroom1 | 书房 | 最符合办公/RTX 3090 工作站假设。 |
| Bedroom2 | 主卧阳台或次卧 | 取决于未来第二台台式机/长期书桌位置。 |
| Bedroom3 | 次卧 | 如果只是上下床卧室，可不设固定网口。 |

这个映射必须由业主确认后再改 `inventory/network_ports.csv` 和施工点位清单。

### 3.3 P2S 位置未画出

旧基线把 Balcony1 作为 Bambu P2S 区。新图显示：

- 次卧阳台有书桌/储物；
- 主卧阳台有书桌/书柜；
- 客厅阳台空间最大，且离生活区近、可做窗板外排，但图上没有 P2S。

目前建议把客厅阳台作为 P2S 首选候选，但必须确认：

- P2S 工作台精确位置；
- 独立插座和备用网口；
- 排风窗板、止回、防虫、物业允许；
- 烟雾、VOC、温湿度传感器位置；
- 噪音、热量和开门通行影响。

### 3.4 洗衣机位置变化

新图中洗衣机位在次卫生间区域。旧清单里的 Balcony2 洗衣/水路候选需要重新审查。

次卫生间应增加或确认：

- 洗衣机专用插座和漏保；
- 洗衣机进水、排水、地漏；
- 水浸传感器可放置位置；
- 是否有烘干机或洗烘套装；
- 是否需要本地排风/除湿；
- 插座和弱电设备必须符合湿区安全距离。

### 3.5 客厅影音方案需要按新图重画

新图客厅下方有长柜，沙发和餐桌位置已画出，且有疑似幕布位置。当前 `PJ-DATA`、`AV-DATA-1/2` 和投影电源不能只按旧文字执行。

必须确认：

- 幕布或电视墙最终位置；
- 投影仪吊装点或短焦/超短焦位置；
- AV 柜/播放器/功放/小交换机位置；
- 影音管路径、管径、两端高度；
- 投影电源、幕布电源和网口是否同图复核；
- 沙发上方/后方是否适合投影，不与空调风口、灯具、吊顶冲突。

## 4. 对各系统的影响

### 4.1 网络端口

需要优先复核这些点：

| 当前点位 | 新图影响 | 建议动作 |
|---|---|---|
| `B1-DATA-1/2` | 应大概率迁移到书房书桌 | 书房桌边保留 2 根 Cat6A。 |
| `B2-DATA-1/2` | 第二固定桌位未明确 | 在主卧阳台/次卧中择一，或保留 1 根空管。 |
| `BAL1-DATA` | P2S 阳台未确认 | 待 P2S 位置确认后重新命名。 |
| `BAL2-DATA` | 洗衣不在阳台的可能性提高 | 若无水路设备，可降级为空管或备用。 |
| `MAC-HOST` | 网络设备间位置未确认 | 不能开工前必须落到平面图。 |
| `RS485-GW` | 起点随网络设备区变化 | RS-485 米数需要重算。 |
| `AV-DATA-1/2` | 需按客厅长柜/幕布重画 | 建议仍保留 2 根 Cat6 或 1 根 Cat6 + 1 根空管。 |

### 4.2 AP 与 Zigbee

新图下的建议仍是两台 AP 起步：

- AP1：过道中部偏左，覆盖书房、厨房、次卧和次卫附近；
- AP2：客餐厅中部或偏右，覆盖客餐厅、主卧、主卧阳台和客厅阳台；
- AP-R：保留空管，等待现场 Wi-Fi 实测；
- Zigbee PoE 协调器：放在过道开放位置，靠近全屋中心，PoE 有线回交换机，不放金属柜内。

需要设计师在天花图上避开风管、检修口、灯具、金属龙骨和中央空调出回风口。

### 4.3 RS-485

RS-485 设计原则不变：日立和格恩分通道、分管、分线。

但因为新图中网络设备区不明确，旧文档中的米数估算暂时只能作为旧基线参考。需要重新确认：

- RS-485 网关实际起点；
- 日立转换器可检修位置；
- 格恩控制器/主板位置；
- 厨房、卫生间、吊顶设备检修口路径；
- 是否存在从衣帽间/玄关到中部吊顶的可行独立弱电管。

在这些点位未确认前，不建议更新最终米数，只建议保留“独立弱电管 + 拉线 + 两端编号 + 封墙前拍照”的施工原则。

### 4.4 用电与插座

新图对强电复核的重点是：

- 书房工作站：桌边独立插座组，考虑 RTX 3090、显示器、外设和网络设备；
- 衣帽间/网络柜候选：若放设备，需 PDU、UPS、散热和检修，不应与冰箱同一封闭空间互相影响；
- 冰箱：图中靠衣帽间/玄关侧，需要独立常电和散热空间；
- 次卫生间洗衣机：湿区专用插座、漏保、水浸；
- 客厅长柜：AV 设备、小交换机、投影/幕布、游戏机等；
- 客厅阳台 P2S 候选：打印机、AMS、排风、传感器需分插座；
- 两个卫生间：智能马桶、镜柜、排风、人体/水浸传感器；
- 窗帘盒：每组窗帘电机侧和 220V 常电需要标明。

智能插座只能作为低风险设备的监测或辅助控制，不应用来切断中央空调、加湿主机、热水器、洗衣机关键运行或 P2S 打印中的主电源。

## 5. 必须向设计师确认的问题

1. 新图中网络设备集中区到底在哪里？是否仍有独立储物/弱电设备柜？
2. 如果放在衣帽间/玄关柜，机柜尺寸、散热、检修、PDU、UPS、配线架、路由器、交换机和 Mac mini 如何布置？
3. 入户光纤/网线从哪里进户，能否直接到网络设备柜？
4. RTX 3090 工作站是否确定放书房？书房桌子最终位置、高度和左右方向是否固定？
5. 是否还需要第二个固定台式机位？是在主卧阳台、次卧，还是其他位置？
6. P2S 确切放哪里？是否确认客厅阳台、排风窗板和物业允许？
7. 客厅幕布/电视/投影/AV 柜最终方案是什么？投影仪是吊装、柜上、短焦还是超短焦？
8. 洗衣机是否最终在次卫生间？是否有烘干机、壁挂洗衣机或其他水设备？
9. 总水阀、净水器、软水机、前置过滤器在哪里？是否需要电动总水阀电源？
10. 日立协议转换器和格恩控制器/主板分别在哪里可检修？
11. 中央空调内机、风口、回风口、检修口会不会与 AP、Zigbee、投影和传感器冲突？
12. 每组窗帘电机在左侧还是右侧？是否双层窗帘？是否需要弱电空管？
13. 入户门高位摄像头和低位门铃是否允许施工？物业和邻里隐私是否有限制？
14. 所有固定柜体完成后，是否还保留可检修弱电管和拉线？

## 6. 建议下一步

1. 先让设计师在新平面图上标出网络设备柜、AP、Zigbee、书房网口、客厅 AV/投影、P2S、洗衣机、水路设备、窗帘电机和 RS-485 起终点。
2. 业主确认房间命名映射：书房、次卧、主卧、次卧阳台、主卧阳台、客厅阳台分别对应旧清单中的哪些编号。
3. 更新 `inventory/network_ports.csv`，把 `Bedroom1/Balcony1/Balcony2/Storage room` 改为与新图一致的中文房间名或双语编号。
4. 更新 `inventory/power_socket_points.csv`，把冰箱、洗衣机、书房工作站、客厅 AV、P2S 和网络柜的插座点位按新图重排。
5. 待网络设备柜、日立转换器和格恩控制器位置确认后，再更新 `docs/current/RS485线路拓扑与米数估算_V1.md` 和 `inventory/rs485_cable_runs.csv`。
6. 在设计冻结前，生成一版“新平面图点位复核 V2”给设计师和施工队确认，避免施工队按旧 Bedroom/Storage/Balcony 文字理解开工。

## 7. 当前不能冻结的事项

- 网络机柜/弱电集中柜位置；
- 书房是否就是 3090 工作站房；
- 第二固定电脑位；
- P2S 阳台和排风路径；
- 客厅投影、幕布、AV 柜的相对位置；
- 洗衣机、烘干机、水路设备最终组合；
- 日立和格恩控制器可检修位置；
- RS-485 最终米数；
- AP、Zigbee、传感器与吊顶/风口/灯具的避让关系。

这些事项确认前，旧清单只能作为设计假设，不能作为施工冻结版本。

# 周末设计师点位图制作基准与优先级（V1）

**日期：** 2026-07-02  
**用途：** 周末给设计师制作强电、弱电、传感器和设备点位图时使用。当前装修队正在拆旧/拆墙，本文件用于水电开槽前锁定不能后补的点位。

## 1. 点位图应该基于什么

本轮点位图必须以以下资料为基准：

1. **主底图：** `references/newFloorPlan.png` 对应的新平面布置图；
2. **现场事实：** 拆旧/拆墙后拍摄的全屋照片和视频，尤其是衣帽间、客厅边吊、空调出风口、入户、厨房、次卫、窗边和弱电箱；
3. **当前施工基线：** `docs/current/设计师发送前点位复核_V1.md`；
4. **强电/插座基线：** `docs/current/用电功率与插座点位复核_V1.md`；
5. **弱电端口基线：** `inventory/network_ports.csv`；
6. **强电插座结构化清单：** `inventory/power_socket_points.csv`；
7. **RS-485 线缆基线：** `docs/current/RS485线路拓扑与米数估算_V1.md` 和 `inventory/rs485_cable_runs.csv`；
8. **施工交底基线：** `docs/current/装修队施工交底清单_V1.md`。

不要再以早期 `V3.1` 户型 PNG 或早期施工 PDF 作为落图基准。早期图可以追溯历史，但新版平面图、衣帽间机柜、次卫洗衣、客厅 AP 偏置和 P2S 衣帽间方案已经覆盖旧基线。

## 2. 设计师需要输出的图纸层级

建议设计师至少输出四张图，避免强弱电和传感器混在一张图上看不清：

| 图纸 | 必须包含 | 目的 |
|---|---|---|
| 强电/插座图 | 固定设备插座、窗帘电机 220V、机柜电源、P2S 插座、投影/幕布、洗衣/厨下/卫浴/水阀电源 | 电工开槽、回路和漏保设计 |
| 弱电/网络图 | Cat6/Cat6A、AP、Zigbee PoE、影音柜、投影、入户门、桌面网口、机柜回线 | 弱电穿管、编号和测试 |
| 天花/吊顶设备图 | AP1、AP2、Zigbee 协调器、投影、幕布、窗帘盒、检修口 | 解决吊顶有限、设备外露和维护问题 |
| 传感器/安全点位图 | 水浸、温湿度、烟感、门窗磁、存在传感器、P2S 监测、机柜温度 | 后续安装和智能中控规划 |

如果设计师只能先做一张总图，至少要用颜色或图例区分：强电、弱电、PoE、RS-485、无线传感器、预留空管。

## 3. 最近几天最高优先级

### P0：水电开槽前必须落图

这些一旦错过，后期补线/补管成本很高：

| 类别 | 必须锁定的点位 |
|---|---|
| 衣帽间设备区 | 12U 优先落地网络机柜位置、门开启、PDU/UPS、通风、弱电总进线、与 P2S/衣物分区 |
| 光纤/主路由 | 联通光纤入户、光猫桥接位置、主路由/防火墙、机柜到光猫路径 |
| 弱电主干 | 所有 Cat6/Cat6A 星形回衣帽间配线架，不串接、不接头 |
| AP | AP1、AP2 点位和网线；不再预留 AP-R；AP2 可放客厅边吊/空调出风口附近可施工区，允许偏置但不能被风管/柜体/灯具遮挡 |
| Zigbee PoE | 中央走廊开放顶面/高位位置 1 根 Cat6，网线暗走、顶面/高位明装，不放金属机柜 |
| RS-485 | 日立空调、格恩中央加湿各自独立弱电管/屏蔽线回衣帽间，不共线、不与强电共管 |
| 客厅影音 | 幕布、投影、影音柜、220V、Cat6、影音穿线管、检修口 |
| 书房/桌面 | 3090 工作站桌位、2 根 Cat6A、多位带地插座、不要被桌腿/柜体遮挡 |
| 主卧阳台/儿童房/书房桌面 | 台灯、电脑、多台电脑候选位置，插座组和必要 Cat6A 网口 |
| 次卫洗衣 | 洗衣机固定位置、插座、防潮/漏保、排水、水浸传感器位置 |
| 厨下/水路 | RO/净水/软水/厨下电源、水浸、总水阀候选、电动阀电源和检修 |
| 窗帘 | 已确认单层；每组窗帘盒 220V、电机侧、检修口、可选弱电空管 |
| 智能开关 | 每个开关盒零线、加深底盒、多控关系 |
| 入户门 | 高位 PoE 摄像头候选、低位门铃 Cat6 或至少空管+拉线 |

### P1：本周最好一起定

这些不一定马上采购，但点位要尽量预留：

| 类别 | 点位 |
|---|---|
| P2S | 衣帽间 P2S 独立台面、独立插座、AMS/耗材空间、可选排风/通风、电源不接机柜 UPS/PDU |
| 机柜扩展 | NAS 预留、第二个 RS-485 网关预留网口/电源；第二个 RS-485 网关未来可直接接交换机和机柜 PDU |
| 未来计量设备 | 电表/水表接入智能系统只做未来可选项；不假设小区正式计费表可本地读取；本轮最多预留空管/拉线条件 |
| 卫浴设备 | 智能马桶、镜柜、排风、浴室柜下电源和防潮检修 |
| 空气/环境 | 温湿度、存在传感器主要安装区；CO2 不作为本轮固定点位，后续用普通插座/USB-C 桌面设备试点 |
| 安全报警 | 独立烟感位置，尤其是客厅/走廊、衣帽间/P2S 附近 |

### P2：可以后装，不阻塞开槽

这些主要是无线或可后期调整，不应拖住施工：

| 类别 | 说明 |
|---|---|
| Zigbee 温湿度 | 多数电池供电，先标安装区域，不需要逐个布线 |
| Zigbee 水浸 | 多数电池供电，但要确认能放到洗衣机、厨下、水阀、加湿设备附近 |
| 门窗磁 | 后期贴装，先标门窗区域 |
| 智能插座 | 后期小批量验证，不给关键设备断电 |
| AP3/AP-R | 本轮取消，不再让设计师为 AP-R 预留空管或网线 |

## 4. 强电图必须优先画什么

强电不是只画普通插座，需要先把固定设备画清楚：

1. 衣帽间机柜：一个独立带地墙插给落地机柜/UPS 输入；UPS 输出接机柜 PDU，供路由、交换机、Mac mini、RS-485 网关等低功耗设备；
2. P2S：一个独立带地墙插或插座组，不接机柜 UPS/PDU，旁边可留排风/通风电源；
3. 3090 工作站：书桌侧多位带地插座，按高功率桌面设备考虑；
4. 客厅影音柜：多位插座、散热、与网口/影音管同点位；
5. 投影/幕布：投影位 220V、幕布电源、影音管路径；
6. 次卫洗衣/烘干候选：湿区合规、防潮、接地、漏保；
7. 厨下净水/RO/水路设备：可检修，不能被橱柜永久封死；
8. 卫浴：智能马桶、镜柜、排风和传感器候选电源；
9. 窗帘盒：已确认单层；每组 220V 常电，按 1 台电机和 1 个控制通道预留；
10. 智能开关：所有开关盒留零线和深盒；
11. 总水阀/电动阀候选：电源、手动操作空间、检修空间；
12. 入户门设备：PoE 优先；若非 PoE，需单独画电源策略。

衣帽间当前强电建议按四个墙插组落图：

| 墙插组 | 用途 | 备注 |
|---|---|---|
| RACK-POWER | 落地机柜/UPS 输入 | UPS 后面可接机柜 PDU；可以给 Mac mini、第二个 RS-485 网关等低功耗设备供电，但不要串接廉价插排 |
| P2S-POWER | P2S/AMS | 独立于机柜 UPS/PDU；打印中不允许普通智能插座断电 |
| VACUUM-CHARGE | 吸尘器/清洁设备充电 | 普通墙插，和 P2S、机柜分开 |
| STORAGE-HIGH-POWER-RESERVE | 高功率/服务备用 | 可作为未来 3090 移位或临时高功率设备候选；如果真接 3090，需电工确认回路容量和散热 |
| P2S-VENT-POWER | 可选排风/通风 | 如果有合适通风路径再做；PLA/PETG 不强制外排 |

## 5. 弱电图必须优先画什么

弱电必须全部星形回衣帽间机柜：

| 编号 | 点位 | 要求 |
|---|---|---|
| AP1 | 左侧中央走廊吊顶/可暗线顶面 | 1 根 Cat6/Cat6A，PoE |
| AP2 | 客厅边吊/空调出风口附近可施工区 | 1 根 Cat6/Cat6A，PoE，允许偏置 |
| ZIGBEE-POE | 中央走廊开放顶面/高位位置 | 1 根 Cat6，PoE；网线暗走，设备明装 |
| B1-DATA | 书房/3090 桌位 | 2 根 Cat6A |
| B2-DATA | 次卧/未来桌位 | 2 根 Cat6A |
| MB-BAL-DATA | 主卧阳台书桌 | 1 根 Cat6A 必选；第二根 Cat6A 或空管作为多电脑/扩展坞备用 |
| AV-DATA | 客厅影音柜 | 默认 2 根 Cat6，受限时至少 1 根 Cat6 + 1 根空管 |
| PJ-DATA | 投影位置 | 1 根 Cat6 备用，另需影音穿线管 |
| ENTRY-H | 入户高位 | 1 根 Cat6，PoE 摄像头候选 |
| ENTRY-L | 入户低位 | Cat6 优先，至少空管+拉线 |
| P2S-DATA | 衣帽间 P2S 台面 | 可选 1 根 Cat6A，P2S 可先用 Wi-Fi |
| RS485-GW | 衣帽间 RS-485 网关 | 网关以太网接入交换机 |

## 6. 传感器图怎么画

传感器图不要误导施工队去给每个 Zigbee 传感器拉线。建议分两类画：

| 类型 | 施工含义 |
|---|---|
| 需要预留供电/管线 | 烟感若选有线型、存在传感器若需 USB/220V、排风/电动阀/窗帘电机；CO2 本轮不专门预留固定点位 |
| 无线/电池安装区 | 温湿度、水浸、门窗磁、普通人体/存在传感器、场景按钮 |

本轮传感器优先标：

- 水浸：次卫洗衣机、厨房水槽/厨下、净水/RO、总水阀、加湿设备附近；
- 烟感：客厅/走廊、衣帽间/P2S 附近；
- 温湿度：客厅、主卧、书房、儿童房/次卧、主卧阳台、卫生间、衣帽间/P2S；安装在远离空调出风口/回风口、窗户直晒和门缝的位置；
- 门窗磁：入户门、阳台门、关键窗；
- 存在传感器：客厅、玄关/过道、书房、卫生间候选；卧室先不强制全屋毫米波。

### 6.1 空调/加湿传感器与独立温湿度点

中央空调和中央加湿自身会在风口/设备侧有传感器，但这些读数更接近设备控制逻辑，不一定代表人体活动高度的真实舒适度。建议每个主要房间仍放一个独立温湿度传感器：

| 房间 | 建议位置 |
|---|---|
| 客厅/餐厅 | 沙发或餐桌附近内墙，离地约 1.2-1.5 m，避开出风口和阳光 |
| 书房 | 书桌附近但不贴电脑主机/显示器热源 |
| 儿童房/次卧 | 床头或书桌附近内墙，避开窗和空调风 |
| 主卧 | 床侧内墙，避开主卧阳台门和空调风 |
| 主卧阳台 | 如果长期作为书桌/阅读区，单独放一个 |
| 衣帽间/P2S | P2S 周边但不贴打印机热源，用于热量/干燥/通风观察 |
| 次卫/主卫 | 干区或门附近，避开直接水汽冲刷；水浸另放地面低位 |

### 6.2 CO2、存在和排风建议

| 类别 | 建议 | 点位 |
|---|---|---|
| CO2 | 可选 1-2 个桌面试点，不作为水电开槽阻塞项 | 不装新风后优先级降低；入住后客厅/餐厅、书房按实际闷热/关门办公情况，用普通插座/USB-C 真 CO2 设备验证 |
| 存在传感器 | 建议先 3-4 个试点，不全屋一次买满 | 玄关/过道、客厅、书房、卫生间；卧室先用门窗/人体/手动场景，避免误触发 |
| 卫生间排风 | 建议纳入强电/开关设计 | 主卫、次卫排风保留实体开关；可后续用智能开关/继电器按湿度和存在延时控制 |
| 衣帽间/P2S 排风 | 只做预留，不作为 PLA/PETG 必选 | 如空间密闭或后期打印 ABS/ASA/PC，再启用低噪声风机/外排；必须有手动开关 |

## 7. 周末给设计师的直接请求

可以直接这样发给设计师：

```text
请基于新版平面图和拆旧后的现场情况，先帮我们出一版强电、弱电、天花设备、传感器/安全点位图。

优先保证水电开槽前不能后补的点位：衣帽间落地机柜、P2S、AP1/AP2、PoE Zigbee、日立/格恩 RS-485、客厅投影/幕布/影音柜、书房/儿童房/主卧阳台桌面网口和插座、次卫洗衣、厨下水路、每个窗帘电机、智能开关零线和入户门预留。AP-R 本轮取消，不再预留。

电表/水表接入智能系统只是未来可选项，不作为本轮必须施工内容；不假设小区正式电表/水表可被业主本地读取。RS-485 本轮刚性只针对日立空调和格恩加湿。电动总水阀默认按强电/水路安全点位处理，预留电源、检修和手动操作空间，不默认接 RS-485。

无线传感器可以先画安装区域，不需要逐个布线；但水浸、烟感、存在传感器若需要供电，请在强电图里标出。CO2 本轮不专门新增固定点位，后续用普通插座/USB-C 桌面设备试点。

所有弱电线必须星形回衣帽间落地机柜，强电/弱电/RS-485/影音管不得混管。封墙、封顶、封柜前需要拍照和编号复核。
```

## 8. 这两天需要现场补充的信息

为了让点位图更准确，拆旧后请尽快拍照/确认：

1. 衣帽间落地机柜和 P2S 能否同时放下，门能否正常开；
2. 客厅边吊/空调出风口实际位置，AP2 是否有合适暗线出线点；
3. 光纤入户和弱电箱位置，能否顺利回衣帽间机柜；
4. 次卫洗衣机、地漏、排水和插座可用墙面；
5. 厨下净水/RO/水阀空间；
6. 投影、幕布、影音柜和检修口关系；
7. 每个窗帘盒是否有足够空间放电机和检修；
8. 日立空调转换器、格恩加湿控制器预计安装位置；
9. 书房/3090 桌位最终方向；
10. 入户门高位/低位预留是否受门套或物业限制。

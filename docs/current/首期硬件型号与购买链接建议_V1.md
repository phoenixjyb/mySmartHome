# 首期硬件型号与购买链接建议（V1）

**日期：** 2026-06-30  
**阶段：** 设计/装修前采购候选；不是最终下单单据。  
**用途：** 把当前已经提到的核心设备收敛成可比价、可采购、可和施工队确认尺寸/供电/点位的候选表。

## 1. 采购原则

1. 先定影响施工的设备尺寸、供电和管线：机柜、PDU、UPS、交换机、AP、Zigbee 协调器、RS-485 网关、投影/幕布管线。
2. 传感器和开关不要一次性买满。先买少量验证 Zigbee2MQTT、信号、外观和电池表现，再批量采购。
3. 采购链接优先使用京东/淘宝搜索页，避免单个商品页失效。正式下单时优先官方旗舰店、自营、带发票和可退换渠道。
4. 所有价格都是 2026-06 设计阶段粗估，装修定稿或下单当天必须重新看成交价。
5. UPS 只给弱电核心：光猫、路由/防火墙、核心交换机、Mac mini/HA 主机、Zigbee/RS-485/MQTT 网关。P2S、3090、空调、加湿、洗衣、冰箱、投影不接机柜 UPS。

## 2. 当前推荐组合

如果现在要按性价比优先推进，我建议先按下面这个组合做预算和点位确认：

| 类别 | 当前首推 | 备选 | 结论 |
|---|---|---|---|
| 网络生态 | TP-LINK / Omada 方向 | UniFi 或 OpenWrt/OPNsense + 独立 AP/交换机 | 国内采购、维护和性价比优先时，先看 TP-LINK |
| 路由/防火墙 | N100/N150 软路由小主机跑 OpenWrt/OPNsense | TP-LINK 企业路由 | 想少折腾可用 TP-LINK；想可控和 WireGuard/策略能力强，用软路由 |

## 6. 暂不建议现在下单的设备

| 类别 | 现在不急的原因 | 现在要做的施工动作 |
|---|---|---|
| 投影仪 | 需要最终幕布尺寸、投射距离、吊装位置、噪音和亮度需求 | 客厅幕布位、投影位、电源、Cat6、影音管、检修口先落图 |
| 门口摄像头/门铃 | 本轮暂不预留 | 后续如确实需要，先确认物业许可、门套结构和隐私边界，再单独评估明装、门锁生态或无线方案 |
| 摄像头/NVR | 需要先定是否真的装室内摄像头 | 只给入口/阳台候选位置留线，不默认全屋摄像头 |
| 全屋智能开关 | 灯光回路、双控、多控、调光、零线和底盒深度未最终落图 | 所有开关盒留零线和深盒，回路图定后再按路数采购 |
| 窗帘电机 | 不急着买电机，但每组窗帘都要先按电动落图 | 每个窗帘盒预留 220V，必要时留弱电空管 |
| 总水阀电动阀 | 需要确认总水阀类型、物业/水工可改性和手动旁通 | 总水阀位置拍照，保留检修空间和可供电条件 |
| NAS/Frigate 主机 | 首期不是施工阻塞项，且摄像头方案未定 | 机柜留空间、网口和非 UPS/UPS 边界 |

## 7. 下单前逐项确认

- 型号后缀和硬件版本是否一致，尤其是 AP、交换机、传感器和 Zigbee 设备。
- 是否支持本地管理和配置导出。
- 是否支持 VLAN/SSID 映射、PoE 控制、端口命名、固定 IP。
- Zigbee 设备的具体 model id 是否在 Zigbee2MQTT supported devices 里。
- Zigbee 协调器优先使用 Ethernet/PoE；不要把 USB dongle 藏在机柜或主机后面作为主方案。
- RS-485 网关是否 4 路独立、隔离、每路串口参数可独立配置。
- UPS 是否有 USB 通讯、可换电池、合适插座类型和足够通风空间。
- 京东/淘宝是否官方旗舰店、自营或可靠店铺；是否支持发票和 7 天/保修。

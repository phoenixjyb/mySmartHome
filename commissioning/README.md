# Commissioning Evidence

This directory will hold site verification and acceptance records.

Planned files:

- `wiring/`: cable labels, test results, and photo index.
- `hvac/`: Hitachi converter settings, room mapping, read-only scan outputs.
- `humidifier/`: Gree/Gen serial settings, read-only scan outputs, write-control validation.
- `zigbee/`: coordinator placement, channel choice, device join records.
- `network/`: VLAN, AP, switch, and UPS validation.
- `safety/`: leak, smoke, UPS, outage, and manual fallback tests.

Evidence should distinguish design assumptions, vendor-confirmed facts, and site-verified results.
